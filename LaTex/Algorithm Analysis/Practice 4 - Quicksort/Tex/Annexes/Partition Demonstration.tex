\subsection{Partition Demonstration:}

Formally demonstrate that {\bfseries\itshape Partition} algorithm has linear complexity.  \hfill \break

{{\bfseries\color{Violet}{Partition}} ( A, p, r ):
\begin{lstlisting}
	x = A[r]
	i = P - 1
	for  j = p to j <= r - 1 do
		if ( A[j] <= x )
			i ++
			exchange ( A[i], A[j] )
	exchange ( A[i + 1] , A[r] )
	return i + 1;			
\end{lstlisting} \hfill
